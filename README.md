---

## Features

- In-game mod menu UI  
- Toggleable mods  
- Customizable keybinds

*(Feature list may change as the project evolves.)*

---

## Requirements

- Gorilla Tag (PC VR)
- Compatible mod loader (e.g. BepInEx)
- Basic understanding of modding GTAG

---

## Installation

1. Install a supported mod loader for Gorilla Tag  
2. Download The Menu - lates releases 
3. Place the mod files into your Gorilla Tag plugins folder  
4. Launch the game and have Fun - O to open and close the Gui
